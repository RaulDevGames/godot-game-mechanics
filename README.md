# Godot Game Mechanics

Una colección de mecánicas de videojuegos desarrolladas en Godot 4.

Cada carpeta corresponde al proyecto final de un episodio de la serie de YouTube.

1. Descarga este repositorio.
2. Abre la carpeta del episodio que quieras.
3. Abre el archivo `project.godot`.
